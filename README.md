# Launchstore

Para executar a aplicação, execute os seguintes passos:

### 1. Subir container do banco de dados
```
docker compose up -d
```

### 2. Iniciar a aplicação
```
npm start
```
> O projeto roda na versão 14.21.3 do node.js
